Submit Fidgets

Add fidgets to an existing group in The Fidget List directory.

Instructions

For each fidget you want to submit, you'll need:

  • Basic information (name, dimensions, weight, etc.)
  • A direct link to an image of the fidget
The image link should be a direct URL that ends with .jpg, .png, or similar. You can right-click on an image and select "Copy image address" to get this link.
